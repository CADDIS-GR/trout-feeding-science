# 🎣 How Trout Feed — 송어 섭식의 과학
> *"먹이가 작을수록 송어는 더 자주 먹습니다."* — Fish the Math

송어의 섭식 행동을 과학적으로 분석하는 인터랙티브 시리즈 아카이브입니다.  
각 에피소드는 독립 HTML 페이지로 구성되며, 차트·인터랙티브 요소·원문 스크립트를 포함합니다.

## 🔑 핵심 개념 요약

| 개념 | 내용 |
|------|------|
| **최적 먹이 크기** | 4~6mm — 항상, 매일, 일 년 내내 |
| **탐색 이미지** | 60초 내 2회 포획 성공 시 형성, 14일~3개월 유지 |
| **깔따구 (Midge)** | 강 수생 곤충의 50% 이상, 연 365일 부화 |
| **송어의 판단 시간** | 2초 이내에 먹이 여부 결정 |
| **추천 훅 사이즈** | #18~#22 |
| **최적 먹이 찾기 이론** | 탐색·처리 비용이 동일할 때 큰 먹이 선호 — 그러나 실제 강에서는 소형 먹이가 압도적 |
| **사회적 학습** | 송어는 동료의 섭식을 관찰해 하루 만에 검색 이미지 형성 가능 |

## 🛠 새 에피소드 추가 방법

1. `ep0N/` 폴더 생성 → `ep03/Trout_Like_Big_Bugs.html` 참고하여 내용 작성
2. 루트 `index.html`의 `.ep-grid`에서 새 카드 추가 (`coming-soon` 클래스 제거)
3. 이전 에피소드 하단 "다음 에피소드" 링크 업데이트
4. 이 README의 에피소드 목록 업데이트
